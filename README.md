# 简历

> This is my resume

[预览](https://lurkerc.github.io/animating-resume/dist/)

## 使用方法

``` bash
git clone git@github.com:Lurkerc/animating-resume.git
cd animating-resume
npm install
npm run dev
```

## 部署方法


1. 编译、上传
    ``` bash
    npm run build
    git add .
    git commit -m "update"
    git push
    ```

2. 开启 GitHub Pages 功能
